什么是 MCP？

Model Context Protocol (MCP) 是一个开放标准，旨在为AI应用程序提供安全、可控的方式来访问外部数据源和工具。它就像是AI助手的"神经系统"，让AI能够感知和操作真实世界的数据和服务。


通过MCP，AI助手不再局限于训练数据，而是可以实时访问数据库、文件系统、API接口等各种资源，从而提供更准确、更及时的服务。

代码示例

# MCP服务器示例
from mcp import MCPServer, Tool, Resource
import sqlite3
import json

class DatabaseMCPServer(MCPServer):
    def __init__(self):
        super().__init__("database-server", "1.0.0")
        self.db_path = "example.db"

    @Tool("query_database")
    async def query_database(self, sql: str) -> dict:
        """执行SQL查询并返回结果"""
        try:
            conn = sqlite3.connect(self.db_path)
            cursor = conn.cursor()
            cursor.execute(sql)
            results = cursor.fetchall()
            columns = [desc[0] for desc in cursor.description]

            return {
                "success": True,
                "data": [dict(zip(columns, row)) for row in results],
                "count": len(results)
            }
        except Exception as e:
            return {
                "success": False,
                "error": str(e)
            }
        finally:
            conn.close()

    @Resource("user_data")
    async def get_user_data(self) -> str:
        """获取用户数据资源"""
        return await self.query_database("SELECT * FROM users LIMIT 100")

if __name__ == "__main__":
    server = DatabaseMCPServer()
    server.run()

# 客户端调用示例
import asyncio
from mcp_client import MCPClient

async def main():
    # 连接到MCP服务器
    client = MCPClient("ws://localhost:8000")
    await client.connect()

    try:
        # 调用数据库查询工具
        result = await client.call_tool("query_database", {
            "sql": "SELECT name, email FROM users WHERE active = 1"
        })

        if result["success"]:
            print(f"查询成功，返回 {result['count']} 条记录:")
            for row in result["data"]:
                print(f"- {row['name']}: {row['email']}")
        else:
            print(f"查询失败: {result['error']}")

        # 获取资源
        user_data = await client.get_resource("user_data")
        print(f"用户数据: {user_data}")

    finally:
        await client.disconnect()

if __name__ == "__main__":
    asyncio.run(main())

技术细节

协议特性

  • 传输层：支持WebSocket、HTTP/2、gRPC等多种传输协议
  • 序列化：使用JSON-RPC 2.0作为消息格式，支持二进制数据传输
  • 认证：支持API Key、OAuth 2.0、JWT等多种认证方式
  • 错误处理：标准化的错误码和错误信息格式
  • 版本控制：向后兼容的版本管理机制

性能优化

  • 连接复用：支持连接池和长连接，减少连接开销
  • 批量操作：支持批量请求处理，提高处理效率
  • 缓存机制：智能缓存常用数据，减少重复请求
  • 流式处理：支持大数据量的流式传输
  • 负载均衡：支持多实例部署和负载均衡
